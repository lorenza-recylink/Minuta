Conectar tu planilla
Pega el link de tu Google Sheet (la pestaña con tus minutas, identificada por el gid). Debe tener acceso "Cualquier persona con el enlace puede ver" para leerse en vivo. El visor reconoce automáticamente cualquier fila que empiece con "Minuta…" como el inicio de una sesión, así que se adapta aunque el formato de fecha o el texto varíen entre una minuta y otra.
Pega el link completo con el gid de la pestaña correcta — la app lo extrae sola.
Sin esto puedes leer y comentar, pero "Guardar cambios" no escribirá en el Sheet — solo podrás exportar.
¿Cómo obtengo la URL del Apps Script?
  1. Abre tu Google Sheet → Extensiones → Apps Script.
  2. Pega el código de AppsScript-Code.gs entregado junto a este visor.
  3. Reemplaza SHEET_ID_AQUI por el ID real de tu planilla.
  4. Implementar → Nueva implementación → tipo Aplicación web.
  5. Ejecutar como Yo, acceso Cualquier usuario → Implementar.
  6. Copia la URL que termina en /exec y pégala arriba.
